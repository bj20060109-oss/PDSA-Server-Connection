Problem Description
In this assignment, you are required to implement a system that simulates a server network.

There are N servers, labeled from 0 to N-1.

Servers may send connection requests to each other.
However, a connection is considered valid only when it is mutual.

That means:

If server a sends a request to server b
And server b also sends a request to server a
Then a valid connection is formed between them
Servers that are connected (directly or indirectly) form a server group.

Your task is to dynamically maintain the server groups and support the required queries.
